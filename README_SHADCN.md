## How to Add Components

### Option 1: Using the CLI (Recommended)
```bash
npx shadcn-ui@latest add button
npx shadcn-ui@latest add card
npx shadcn-ui@latest add input
```

### Option 2: Manual Installation
1. Visit https://ui.shadcn.com/docs/components
2. Click on any component you want
3. Copy the code
4. Create the component file in `src/components/ui/`
5. Import and use it!

## Example Usage

```jsx
import { Button } from "./components/ui/button"

function App() {
  return (
    <Button>Click me</Button>
  )
}
```

## Notes

- Components are copied into your project (not installed as dependencies)
- You can customize them however you want
- All components use Tailwind CSS
- Dark mode is supported via the `dark` class
